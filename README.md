### Project

Building a Url shortener similar to (bitly.com)

packages used:

- Node js (express)
- ejs (for template)
- mongoose (to link to monogo db)
- shortid ( to generate random ids)

## Install

```sh
npm install
```

## Run

```sh
npm run devStart
```
